# 🌸 Period & Cycle Tracker

A beautiful, private, and easy to use period & cycle tracker — no account, no data sharing, everything stays on your device. Built with love 💙

👉 [View it here](https://minajuddin0510.github.io/period-cycle-tracker)

## ✨ Features

- 🗓️ **Cycle Calculator** — enter your last period date and cycle length to get instant predictions
- 📊 **Cycle Summary Cards** showing:
  - 🔴 Next period date with countdown
  - 🥚 Ovulation & fertile window
  - ⚠️ PMS window
  - 🌙 Current cycle phase (Menstrual / Follicular / Ovulation / Luteal)
  - 💚 Current cycle day
- 🗓️ **Color Coded Monthly Calendar**:
  - 🔴 Period days
  - 🟠 PMS days
  - 🟢 Ovulation/fertile days
  - 🩷 Today highlighted with a glow
- 😣 **Symptom Logger** — track cramps, headaches, fatigue, mood swings, nausea & bloating by date
- 📜 Last 7 days symptom history
- 💬 **Daily Wellness Quote** fetched from ZenQuotes API
- 🔒 100% private — all data saved locally on your device, never sent anywhere
- 📱 Fully responsive — designed mobile first

## 🛠️ Built With

- HTML5
- CSS3 (glassmorphism, soft glow animations)
- Vanilla JavaScript
- [ZenQuotes API](https://zenquotes.io) — free, no key needed
- [Poppins Font](https://fonts.google.com/specimen/Poppins) — Google Fonts

## 🔑 API Key

None needed! ZenQuotes is completely free and open. No signup, no credit card, nothing. ✅

## 🔒 Privacy

This app stores all your data **locally on your device** using localStorage. No data is ever sent to any server. Your cycle information is completely private. 💙

## 🚀 Getting Started

### Option 1 — Open locally
```bash
git clone https://github.com/minajuddin0510/period-cycle-tracker.git
cd period-cycle-tracker
open index.html
```

### Option 2 — Live on GitHub Pages
1. Push `index.html` to your GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Live at `https://minajuddin0510.github.io/period-cycle-tracker` 🎉
